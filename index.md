## Detection Content
 - [Splunk Detection Research](https://research.splunk.com/detections/) - 1170 (at time of writing) Splunk Analytics built to find evil. Great detection repository.

## Offensive Security
- [PIMPMyKali / GitHub](https://github.com/Dewalt-arch/pimpmykali) - Run on new Kali VMs. This will configure alot of the items required for functionality (E.g. fix the python module issue, etc).

- [Impacket / GitHub] (https://github.com/SecureAuthCorp/impacket) - Impacket is a collection of Python classes for working with network protocols. Tools in this suite are commonly used by attackers to execute code/commands on remote systems.
